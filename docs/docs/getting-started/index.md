# Getting started

This guide takes you from installing `xdrun` to running useful project automation. Follow the pages in order:

1. [Install `xdrun`](install.md) on Linux, macOS, or Windows.
2. [Enable shell autocomplete](autocomplete.md) for tasks, built-in commands, and flags.
3. [Initialize a spec](initialize.md) in normal or minimal mode.
4. [Run tasks](run.md) and pass parameters.
5. [Start from a project template](templates.md) for common frameworks and toolchains.

If something does not work as expected, see [Troubleshooting](troubleshooting.md).
